“展示天合风采 共建书香社区”汇报演出
2016-01-19 2 min read # 天合 # 书香 # 文艺 # 汇报 # 演出

2016年1月19日下午1点，在寒潮到来之前天合社区组织居民在小区广场隆重举行“展示天合风采　共建书香社区”汇报表演。由天合家园、水尚阑珊、天成小区选送的15支演艺队伍，经过一年时间的学习训练给大家呈现了一场精彩的演出，节目有歌曲、舞蹈、戏曲、太极拳等。整场演出节目丰富多样、连接紧凑、气氛热烈，活动期间还穿插了社区优秀志愿者表彰、有奖知识问答、抽奖等环节，活动气氛热烈。
