天合社区开展健康养生大讲堂

2016-03-11 1 min read # 健康 # 养生 # 大讲堂 # 天合社区

一日之计在于晨，一年之计在于春；阳春三月，万物复苏，四季之中春季是体质投资的最佳季节。2016年3月11日下午1点半，庄桥街道社区卫生服务中心的三位医生来到天合社区会议室，给辖区退休工人、老年党员开展了一次健康养生大讲堂活动。
首先，由欧医生给大家讲了题为《老年人饮食与健康》的讲座，欧医生从老百姓日常饮食出发，详细阐述了老年人在饮食方面的诸多注意事项。然后由徐医生给大家讲了题为《老年人心理健康》的讲座，鲜活的案例分析，在场的听众不时发出赞同的声音。最后由应医生给大家讲了题为《冬春季呼吸道疾病的预防与诊治》，随着空气污染越来越严重，各类呼吸道疾病也时常发生在自身周围，居民对这个话题非常关注，认真听讲。
通过此次讲座，进一步提高了辖区老年人的自我保健意识，促使他们在养成良好的卫生饮食习惯的同时，还促使他们注重自身的心理卫生健康问题，以强健的体魄、健康的心理、丰富的疾病预防知识，享受幸福安康的晚年生活，为建设美好社区发挥余热。
