天合“蒲公英公益亲子阅读社”参加江北区优秀社区社会工作项目展示

2016-03-17 1 min read # 蒲公英 # 阅读社 # 社会工作 # 江北 # 公益 # 亲子 # 天合

为做好“国际社工日”主题宣传活动，弘扬社会工作精神，传播社会工作知识，展示江北区社会工作者风采。2016年3月17日下午，在江北区文化中心，区民政局举行第十个“国际社工日”庆祝活动。活动有三项议程：第一项是戴瑜副区长致辞；第二项优秀社区社会工作项目展示；第三项是社区文艺节目展演。
“优秀社区社会工作项目展示”作为此次庆祝活动的重要环节，一共有9个社区社会组织参与了此次项目展示。天合社区的“蒲公英公益亲子阅读社”经过多次培训和层层筛选之后，终于在今日登上舞台，向全区近300名持证社工介绍“蒲公英公益亲子阅读社”的项目情况与发展计划。下一步天合社区将继续协调社会资源、招募和培训志愿者、通过丰富多彩的亲子阅读活动，结合自媒体与传统媒体的推广，将“蒲公英公益亲子阅读社”打造成更具影响力和知名度的社区社会组织。
