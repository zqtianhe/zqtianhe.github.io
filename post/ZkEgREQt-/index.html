江北新华书店进入天合社区亲子公益活动

2016-02-21 1 min read # 江北 # 新华书店 # 公益 # 亲子 # 天合社区

寒假只剩个尾巴了，为了丰富小朋友们的寒假生活，天合社区居委会特别邀请江北新华书店来到长岛花园，给社区的小朋友们带来一场精彩的公益绘画活动。2月21日上午9:30，长岛花园物业服务中心活动室里人头攒动，来自社区的十几位小朋友围坐在课桌旁，听小猴老师讲如何画一只可爱的猴子。
小猴老师真名李卉，原名温乘心，国画家，温淑静、温淑怡姐妹的侄孙女，师从湖州著名女画家陈伏寅老师，学习国画24载有余。在猴年春节，由小猴老师教小朋友们画猴子，显得特别有意义。老师主要讲述了如何一步步用蜡笔画一只可爱的猴子、还有草地、天空和树枝。然后让小朋友们拿起画笔在白纸上练习起来，老师还在一旁耐心指导。不一会一幅幅充满童趣的画作展示在大家面前，小朋友们对自己的作品非常满意。
据悉，这是江北新华书店首次走进社区。书店的张老板说，以后还将推出更多形式的活动，给社区居民带来更多精神享受和快乐心情。
