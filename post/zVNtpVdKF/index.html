天合社区党员学雷锋志愿服务活动

2016-03-03 1 min read # 党员 # 雷锋 # 志愿服务 # 天合社区

为弘扬社区党员的志愿服务精神，将学雷锋的精神落到实处，营造社区党员参与志愿服务活动的良好氛围，3月3日下午，在天合家园一期南大门组织开展了一场“天合社区党员学雷锋志愿服务活动”。党员学雷锋志愿服务队提供的服务内容有生育咨询、量血压测血糖医疗服务、法律司法咨询、银行理财咨询、理发等等，吸引了许多途经的居民参与活动。
值得一提的是天合社区的徐社工，作为一名预备党员，也积极报名参加此次学雷锋活动。结合自己分管计划生育工作时积累的知识，现场给居民提供二胎政策、独生子女政策、生育健康知识和二胎免费健康检查卡的信息咨询服务，并向居民发放生育健康知识读本，宣传国家生育政策。党员志愿者们的贴心服务赢得了居民的广泛好评。
通过此次组织社区党员学雷锋活动，旨在引导更多的人投身于社区志愿服务之中，争做一个“雷锋式”的好人，参与志愿服务，为构建和谐社区共出一份力。
