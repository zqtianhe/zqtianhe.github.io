《春天故事会》蒲公英公益亲子阅读活动

2016-03-20 2 min read # 故事会 # 阅读 # 蒲公英 # 春天 # 公益 # 亲子

春天。枯黄的原野变绿了。新绿的叶子在枯枝上长出来。阳光温柔地对着每个人微笑，鸟儿在歌唱飞翔。花开放着，红的花，白的花，紫的花。星闪耀着，红的星，绿的星，白的星。蔚蓝的天，自由的风，梦一般美丽。
每个人都有春天。无论是你，或者是我，每个人在春天里都可以有欢笑，有陶醉。

2016年3月20日上午，天合社区志愿者猫猫老师带领10几位小朋友们来到长岛花园活动室，给大家带来一场以春天为主题的公益亲子阅读活动。这是长岛花园活动室装修之后第一次使用，里面还有一点气味，社工们早早地就来到活动室打开门窗通气，并挂好横幅布置活动场地。活动开始后主要有下面几个流程：暖场音乐《春天在哪里》——分享绘本故事1《遇见春天》——创意手工“飞吧，小风筝”——律动音乐《小跳蚤》——分享绘本故事2《999个青蛙兄弟大搬家》。通过猫猫老师的生动讲述，将小朋友们带到了一个又一个新奇的童话世界里。参加活动的小朋友听得很认真，还在家长的帮助下完成了风筝绘画。
