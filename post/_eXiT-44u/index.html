天合社区：开展消防安全知识培训

2016-01-27 2 min read # 消防 # 安全 # 知识 # 培训 # 天合社区

为确保辖区居民过一个平安祥和的春节，保障人民群众生命财产安全，确保大家做到安全用电、用气、用火，预防和减少各类火灾事故的发生，营造良好的社区消防安全氛围，1月27日下午1点半，天合社区邀请市消防中心房老师前来我社区开展了一场“消防安全知识培训”的讲座。
在消防安全知识讲座上，房老师引用了大量发生在宁波本地的真实火灾事例，结合血淋淋的图片和视频为物业公司代表和居民讲解了防火的重要性，为大家上了一堂发人深省的消防安全常识课。当看到视频里的人从高楼跳下，当听到房老师讲到消防通道被私家车堵住时，引发了热烈地讨论。讲座围绕火灾的防范、扑救和自救等知识、技能来展开，重点介绍了日常用火与日常用电方面的安全防火知识，以及火灾发生时的逃生技能。　　　　
此次讲座提高了物业公司和居民的消防安全意识，确保广大辖区居民生活平安、幸福，使广大居民们能够度过一个愉快祥和的新春佳节。
