天合义工参加庄桥青年义工学雷锋志愿服务活动

2016-03-03 1 min read # 义工 # 庄桥 # 学雷锋 # 志愿服务 # 天合

为进一步弘扬雷锋精神，广泛开展向雷锋同志学习的活动，在全国第53个学雷锋日到来之际, 庄桥街道团工委组织开展了《“义”起来·更精彩　庄桥青年义工学雷锋志愿服务活动》。2016年3月3日上午，在庄桥街道河西社区的舞林广场上，义工们组成了一支支志愿服务小分队，有配钥匙、理发、量血压、测血糖、检查口腔、理财投资咨询、列车交通等服务内容。
为响应街道团工委向雷锋同志学习的号召，天合社区派出了拜尔口腔的义工李医生，为广大居民免费检查牙齿，解答居民在牙齿健康方面遇到的问题，宣传口腔卫生、牙齿保护方面的知识；此外还派出了义工参加了捡垃圾小分队，清理舞林广场周边的垃圾死角，美化周围环境。并在活动结束以后，归还桌椅，扫除理发活动留下的头发。
家住河西社区的居民们都纷纷过来参与活动，配钥匙、排队理发、检查牙齿、咨询理财……义工们的雷锋精神值得称赞，希望雷锋精神能这样一年一年传承下去，让更多的人参与到志愿服务当中来，帮助需要帮助的人。
