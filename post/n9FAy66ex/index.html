天合社区《请把我埋葬在春天里》公益绘画培训活动

2016-03-08 1 min read # 春天 # 绘画 # 公益 # 培训 # 天合社区

严冬已逝，春天悄悄地来到了人间。春天，是美好的季节，像一位神奇的能工巧匠，将人间雕琢在花红柳绿的景色里，给平凡的生活赋予了新意。写生是直接进行描绘的一种绘画方法，可以培养观察和感知的能力；在比较短的时间内概括地画出形象，可以提高作画注意力；边看物象、边画、边提炼概括，能有效地使眼手脑的配合能力协调发展。
2016年3月6日下午，在年轮文化书店里，天合社区联合江北新华书店邀请美集印象的小猴老师给居民上了一趟以春天为话题的写生水彩绘画培训课。本次《请把我埋葬在春天里》课程有三个环节，小猴老师讲解了写生基础和写生技巧，然后让大家拿起画笔跟着老师走进春天实战写生。
此次活动有一些小朋友的家长通过社区的微信公众号得知了活动消息，在周末带着孩子前来参加，通过此次培训活动学习了绘画技巧、交到了志趣相同的朋友。
