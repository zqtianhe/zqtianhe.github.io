天合社区开展“金海豚”益智闯关活动

2019-08-18 1 min read # 金海豚 # 益智 # 闯关 # 亲子 # 天合

为了增进社区居民的亲子感情、促进亲子的相互理解、提高家庭成员之间的默契和沟通，天合社区联合金海豚教育培训机构于2019年8月18日上午开展了“金海豚”益智闯关活动。本次活动主要是加强家长与子女之间的情感交流、让孩子们在游戏中学习知识、开发潜能，同时增加居民与社区之间的沟通。
组织开展此次益智闯关活动，让小朋友与家长一起体验《谁的反应快》、《数字去哪儿了》、《我是小小农场主》、《合作串珠》等游戏，增进孩子、家长、同伴之间的感情，在游戏中体验快乐、学习知识，实现孩子健康和谐成长。
每位小朋友通过自己的努力都获得了一份礼品，通过此次活动给小朋友们和家长留来了深刻的美好印象。天合社区以推进共建单位良性互动， 打造温暖幸福社区，提高社区居民幸福指数为目标，拉进居民与社区之前的距离。
